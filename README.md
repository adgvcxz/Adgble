# Dribbble-MVVM
Dribbble Android MVVM

用MVVM写的Dribbble简易客户端

[Retrofit](https://square.github.io/retrofit) ,  [RxJava](https://github.com/ReactiveX/RxJava)

采用数据绑定完成日间夜间主题切换

双向绑定等
